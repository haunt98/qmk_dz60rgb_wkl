# QMK keymap for dztech/dz60rgb_wkl

## Install

Run `make` to build the firmware.

If success, use `dztech_dz60rgb_wkl_v2_1_haunt98.bin` to flash with [QMK Toolbox](https://github.com/qmk/qmk_toolbox).
